Waste-Management-System

This is a sustainable waste management project. The supply chain of waste is monitored over every step, from production to dumping. Waste collection process has been made effective via geo-tagging process and path planning algorithms.

Team members: Aman Saraf, Aman Goyal, Ankur Khosla, Ashwin Goyal and Dev Pardesi Institution: Indian Institute of Technology (IIT) Ropar. Project Supervisor: Prof. Harpreet Singh, IIT Ropar
